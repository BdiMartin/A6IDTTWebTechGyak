A CSS (angolul Cascading Style Sheets) a számítástechnikában egy stílusleíró nyelv, mely HTML vagy XHTML típusú struktúrált dokumentumok megjelenését írja le.


A CSs elnevezésénél a Cascade szó rangsorolásra utal, tehát nem mindegy, hogy az adott stilus hol helyezkedik el a rangsorban. Az ütközések elkerülése végett még pár egyszerű szabály létezik. Az öröklés az XHTML elemek egymásba ágyazottságán alapul, így az egymásba ágyazott elemek öröklik az őket tartalmazó elem beallitásait. A szükitėsnek a stilusok sulyozásánál van szerepe. Azaz ha meg akarjuk különböztetni egy stilus fontosságát, akkor szükebben, vagyis pontosabban kell megadnunk azt, így vehetjük rá a böngészőt, hogy használja. A stílusok elhelyezkedésüktől fügEõen fontossagi sorrendben állnak: Böngészőstílus, Beágyazott stílus, szövegközi stílus.
